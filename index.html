Processing for Android allows you to create Android apps using Processing. You can run your Processing sketches on Android devices (phones, tablets, smartwatches) with little or no changes in the code, and without having to take care of tasks such as installing SDK files, creating project folders, or editing layout files. You can also access the Android SDK to get sensor data and other advanced features, and export your sketch as a signed package to upload to the Google Play Store.

The current 3.0 version of the mode lets you to develop regular Android apps. The upcoming 4.0 version will add support for live wallpapers, watch faces, and Cardboard. Stay tuned for updates from the Contribution Manager in the Processing environment, or Processing's twitter channel!
