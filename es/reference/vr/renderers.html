Motores de renderizado en VR

La biblioteca VR que se está incorporada en el modo Android por defecto incluye dos motores de renderizado: STEREO y MONO. El motor STEREO genera una vista estéreo del bosquejo, dibujando cada cuadro dos veces (una vez para cada ojo) y aplicando las transformaciones oculares correspondientes.


import processing.vr.*;

void setup() {
  fullScreen(STEREO);
}

void draw() {
  translate(width/2, height/2);
  box(300);
}

Meanwhile, the MONO renderer will generate a monoscopic view, still reflecting the movement of the phone.


import processing.vr.*;

void setup() {
  fullScreen(MONO);
}

void draw() {
  translate(width/2, height/2);
  box(300);
}
