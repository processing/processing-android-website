VRCamera

El objecto VRCamera ofrece algunas utilidades para facilitar la configurión de la vista en VR.

  • sticky ()

    Establece las coordenadas para el sistema centrado en el punto de vista, por lo que es más fácil dibujar elementos que son estáticos con respecto al movimiento de la cabeza.

  • noSticky ()

    Reinstaura el sistema global por defecto, eliminando cualquier transformación que pueda haberse aplicado en el espacio centrado en el punto de vista.

  • setPosition (flotante x, flotante y, flotante z)

    Establece la posición de la cámara en el punto (x, y, z).

  • setNear (flotante cerca)

    Establece la distancia Z al plano cercano.

  • setFar (flotante lejos)

    Establece la distancia Z al plano lejano. 


import processing.vr.*;

VRCamera cam;

void setup() {
  fullScreen(VR);
  cameraUp();
}

void draw() {
  background(200, 0, 150);
  cam.setPosition(0, 0, 400);
  //...
  cam.sticky();
  translate(0, 0, 200);
  circle(0, 0, 50);
  cam.noSticky();
}
