eyeType

Esta variable indica qué está siendo dibujando por el motor de renderizado VR. Será IZQUIERDA o DERECHA cuando se usa el motor VR/STEREO, o MONOSCOPIC cuando use el motor MONO.


import processing.vr.*;

void setup() {
  fullScreen(VR);  
}

void draw() {
  PGraphicsVR pvr = (PGraphicsVR)g;
  if (pvr.eyeType == PVR.LEFT) {
    background(200, 50, 50);
  } else if (pvr.eyeType == PVR.RIGHT) {
    background(50, 50, 200);
  } else if (pvr.eyeType == PVR.MONOCULAR) {
    background(50, 200, 50);
  }
}
