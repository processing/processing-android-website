Pantallas especiales

La pantalla del reloj puede usar modo ambiente de pocos bits, o protección contra quemado, los cuales podemos verificar con las siguientes funciones:


void setup() {
}      

void draw() {
  if (wearLowBit()) {
    // ...
  } 
  if (wearLowBit()) {
    // ...
  }  
}


                
De acuerdo a la documentación de Google:
                
    
                   
  • En el caso de dispositivos que incluyen modo ambiente de bit bajo, la pantalla soporta menos bits para cada color en el modo ambiente, de tal forma que el programados puede deshabilitar el suavizado y el filtrado de mapa de bits cuando el reloj entra en el modo ambiente.
    • 
                   
  • En dispositivos que requieren protección contra quemado, se recomienda evitar el uso de bloques de píxeles de gran tamaño en el modo ambiente, y no colocar dibujos a menos de 10 píxeles del borde de la pantalla, porque el sistema mueve los contenidos de la pantalla periódicamente para evitar quemado de píxeles.
