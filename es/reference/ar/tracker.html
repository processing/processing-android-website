ARTracker

Se necesita al menos un objeto ARTracker en el bosquejo para manejar todas las entidades traqueables (solo planos en este momento) detectadas en la escena.

  • start()

    Inicia el seguimiento.

  • stop()

    Detiene el seguimiento.

  • count()

    Devuelve el número actual de planos traqueables en la escena.

  • ARTrackable get(int i)

    Devuelve el i-ésimo plano traqueable.

  • ARTrackable get(String id)

    Devuelve el traqueable con el ID dada.

  • ARTrackable get(int mx, int my)

    Devuelve el traqueables que está siendo seleccionado por el puntero táctil.

  • clearAnchors(Collection anchors)

    Libera los anclajes en la lista que ya no se estan siendo seguidos y los elimina de la lista. 


import processing.ar.*;

ARTracker tracker;
ARAnchor anchor;

void setup() {
  fullScreen(AR);
  tracker = new ARTracker(this);
  tracker.start();  
}

void draw() {
  for (int i = 0; i < tracker.count(); i++) {
    ARTrackable trackable = tracker.get(i);
    // ...
  }
}
