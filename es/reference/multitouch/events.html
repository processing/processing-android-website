Eventos táctiles

Cada vez que se detecta un nuevo puntero táctil, Processing genera una serie de eventos que permiten seguir el inicio, cambio de posición, y final de un toque.


void setup() {
  fullScreen();
}

void draw() {
  if (touchIsStarted) {
    background(150);
  } else {
    background(255);
  }
}    

void touchStarted() {
  println("touch started");
}

void touchMoved() {
  println("touch moved"); 
}

void touchEnded() {
  println("touch ended");  
}
