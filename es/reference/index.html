API de Android

La clase PApplet en el modo Android brinda algunas funciones de la API de Android subyacente a efectos de facilitar el maneko de notificaciones del sistema:

onActivityResult()
onNewIntent()

La API de fragmentos para agregr ítems de menu la barra del app también es disponible a través de PApplet:

onCreateOptionsMenu()
onOptionsItemSelected()
onCreateContextMenu()
onContextItemSelected()
setHasOptionsMenu()
