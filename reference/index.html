Almost all of the Processing API can be used in the Android mode. There is a number of new functions and constants that are unique to the Android mode in version 4.0+, specially relevant to wallpapers, watch faces, and VR apps.

Android API

The PApplet class in the Android mode also exposes some functions from the underlying Android API to make easier to receive updates from the system:

onActivityResult()
onNewIntent()

The Fragment API that allows adding menu items to the App Bar is also available through PApplet:

onCreateOptionsMenu()
onOptionsItemSelected()
onCreateContextMenu()
onContextItemSelected()
setHasOptionsMenu()
