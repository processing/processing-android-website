Cardboard renderers

The Cardboard library includes two renderers: STEREO and MONO. STEREO will generate a stereo view of the sketch, rendering each frame twice (once for each eye) and applying the approriate eye transformations, while MONO will generate a monoscopic view, still reflecting the movement of the phone.


import processing.cardboard.*;

void setup() {
  fullScreen(PCardboard.STEREO);
}    


import processing.cardboard.*;

void setup() {
  fullScreen(PCardboard.MONO);
}
